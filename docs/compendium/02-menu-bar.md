# The Menu Bar: Global Asset and Environment Configuration

The Menu Bar, located at the uppermost region of the editor window, provides foundational controls for file management, window layout, and editor-specific preferences.

---

## File Menu

Governs serialization and I/O operations of the material asset.

| Command | Description | Shortcut |
|---------|-------------|----------|
| **Open Asset** | Launch Global Asset Picker to locate other assets | — |
| **Save** | Write current asset to disk (.uasset) | `Ctrl+S` |
| **Save As** | Create duplicate with new name | — |
| **Save All** | Save all dirty assets project-wide | `Ctrl+Shift+S` |
| **Choose Files to Save** | Granular selection dialog | — |

> [!TIP]
> "Save" secures data to disk. "Apply" updates the render cache. Both are needed for complete workflow.

---

## Edit Menu

Handles history stack and deep editor configuration.

| Command | Description | Shortcut |
|---------|-------------|----------|
| **Undo** | Revert last graph change | `Ctrl+Z` |
| **Redo** | Restore reverted change | `Ctrl+Y` |
| **Editor Preferences** | Global UE configuration | — |

### Mali Offline Compiler Setup

For mobile optimization, configure external tools via **Edit → Editor Preferences**:

1. Locate `malisc.exe` from the Mali Offline Compiler installation
2. Set the path in Editor Preferences
3. Platform Stats panel will now show accurate Arm Mali GPU cycle counts

---

## Asset Menu

Context-aware tools for project hierarchy management.

| Command | Description |
|---------|-------------|
| **Find in Content Browser** | Sync browser to current asset location |
| **Reference Viewer** | Visual dependency graph |

### Reference Viewer

Displays a visual graph showing:

- **Upstream**: Textures and functions this material consumes
- **Downstream**: Actors, blueprints, and materials that reference this asset

> [!CAUTION]
> Always check Reference Viewer before deleting assets to prevent broken references.

---

## Window Menu

Layout manager for toggling panel visibility.

| Panel | Purpose |
|-------|---------|
| **Graph** | Main node editing canvas |
| **Viewport** | 3D preview window |
| **Details** | Property configuration |
| **Palette** | Available node list |
| **Stats** | Performance metrics |
| **Find Results** | Search results for graph nodes |
| **Preview Scene Settings** | HDRI, lighting, post-processing |
| **HLSL Code** | Raw shader code viewer |
| **Shader Code** | Platform-specific shader text |

### HLSL Code Viewer

Opens a text viewer displaying the raw High-Level Shading Language code generated by the graph—indispensable for optimizing instruction count.

### Shader Code Viewer

Access platform-specific shader text:

- DirectX SM5
- Vulkan SM5
- Metal
- DirectX ES 3.1
